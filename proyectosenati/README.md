# Sistema Web de Gestión de Ventas e Inventario Tío Guston
El Sistema Web de Gestión de Ventas e Inventario es una herramienta diseñada para facilitar el control y la administración de las operaciones comerciales de una empresa.
Permite registrar ventas, gestionar productos y llevar un seguimiento preciso del inventario en tiempo real.

## Descripción del negocio
Nombre: Polleria Tio Guston <br>
Contexto:Una pollería es un negocio dedicado a la venta de pollo preparado, ya sea a la brasa, frito o en distintas presentaciones, acompañado generalmente de guarniciones como papas fritas, ensaladas y bebidas.
Este tipo de establecimiento suele tener una alta demanda diaria, especialmente en horarios de almuerzo y cena.
Justificación: La Pollería Tío Gusto gestiona sus ventas e inventario de forma manual mediante cuadernos y hojas de cálculo, lo que genera errores frecuentes, pérdida de información y dificultad para conocer el stock real de insumos en tiempo real.
